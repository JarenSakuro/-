阮鸿视界（moskita.online）
背景与目标
“阮鸿视界”是一个以强视觉叙事与交互引导为核心的个人/品牌展示网站，设计灵感参考“影视飓风”官网（ysjf.com）的沉浸式体验：以全屏无声视频开场、清晰的导航入口、向下滚动的叙事节奏，以及以照片内容为核心的时间线/轮播叙事模块。

网站部署在宝塔 Linux 环境（Nginx 1.22.1 + PHP 8.2.28 + MySQL 5.7.42 + phpMyAdmin 5.1），域名为 moskita.online。站点内容需要通过浏览器后台由单一管理员维护（新增/编辑相册、照片、标签、排序、新闻链接、简介单页等），媒体文件上传存放在服务器本地 uploads 目录。

用户与角色
访客：浏览开场视频、查看时间线轮播叙事、进入照片库按相册浏览照片、查看简介单页、跳转外部新闻来源。
管理员（单人）：通过 /admin 登录，管理网站内容（相册、照片、标签、时间线条目、外部新闻链接、简介单页等）。
核心信息结构（前台）
首页
全屏无声开场视频（当前阶段为固定文件路径），用户点击视频或任意按键后进入主视觉状态（动态视觉标签/引导层）。
左上角四个导航入口：
“河北阮鸿”：艺术化大字标题，作为品牌标识/回到首页。
“阮鸿简介”：跳转至单篇文章页（风格简洁平静）。
“阮鸿视界”：进入照片库（以相册为核心）。
“阮鸿新闻”：外部新闻转载链接聚合页（点击直接跳转源站）。
向下滚动的叙事段落：
视频下方浅色英文日期标记。
轮播图 + 时间线叙事模块：左右箭头切换，点击摘要进入详情页。时间线条目以“照片”为核心（照片属于相册/标签，带描述）。时间线主要按“日期（年月日）”自动排序；允许同一天多条，并支持“同日内手动排序”。
“阮鸿素材库”：以大小错落的布局展示多个照片集合（相册卡片网格/拼贴式布局），可进入相册详情。
照片库（阮鸿视界）
以“相册/模块”为主（例如“荣誉奖杯”“团队温暖”等）。
相册支持：封面图、简介（可选）、手动排序。
照片支持：上传原图、生成/保存缩略图（实现方式后续确定）、描述、所属相册、标签（多选）、拍摄日期（年月日，手动填写）。
相册内照片列表页默认按“拍摄日期（年月日）”自动排序展示。
标签用于跨相册筛选/聚合。
照片详情页
展示核心信息：大图、描述、所属相册、标签。
支持扩展浏览与发现：
“同相册更多照片”（引导继续浏览该相册）。
“按标签跳转/筛选”（点击标签进入标签聚合/筛选视图）。
新闻聚合（阮鸿新闻）
管理员维护外部新闻链接：标题、来源名、跳转 URL、发布日期（可选）、封面图（可选）。
前台点击直接跳转到源站（站内不做转载正文）。
简介页（阮鸿简介）
单篇文章内容，由管理员在后台可编辑（富文本）。
后台管理（/admin）
单管理员账号（用户名+密码），密码加密存储。
内容管理范围：
相册：增删改查、封面图上传/选择、手动排序。
标签：增删改查。
照片：上传、编辑描述、绑定相册与标签、维护拍摄日期（到天）、上/下架（若需要）。
时间线条目：以“照片”为条目来源/引用（选择一张照片并配置在时间线中显示的标题/摘要/日期等）；前台按日期自动排序，同日内支持手动排序。
外部新闻链接：增删改查、排序。
简介单页：编辑与发布。
约束与非目标（当前阶段）
开场视频路径先固定写死，不在后台更换。
仅一个管理员账号，不做多角色权限体系与审计。
媒体文件存储在服务器本地 uploads 目录，不接入对象存储。
重点在“相册 + 标签 + 排序”的照片内容体验与首页时间线叙事呈现。
